This library was generated with [Nx](https://nx.dev).

## Running unit tests

Run `nx test helseid` to execute the unit tests via [Jest](https://jestjs.io).

Configuration variables:

* HELSEID_AUTHORITY
* HELSEID_CLIENT_ID
* HELSEID_CLIENT_SECRET
* HELSEID_JWKS
* HELSEID_REDIRECT_URIS
* HELSEID_SCOPES
